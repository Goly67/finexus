📋 Setup Guide

Follow these steps to fully connect Finexus:

  1. Create a Firebase Project
    Go to console.firebase.google.com → New Project → Finexus
  2. Enable Google Authentication
    Build → Authentication → Sign-in method → Google → Enable
  3. Create Firestore Database
    Build → Firestore Database → Create → Start in test mode
  4. Get Config Keys
    Project Settings → Your Apps → Web → Register App → Copy config
  5. Update Both HTML Files
    Replace YOUR_API_KEY and other placeholders in both index.html and admin.html
  6. Add Google Drive Images
    Upload image to Google Drive → Share → Anyone with link → Copy URL
    Paste the URL into the Image field when adding a property. It auto-converts!
  7. Set Firestore Rules (Production)
    rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /properties/{propId} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
