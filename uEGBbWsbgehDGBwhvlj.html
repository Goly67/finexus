Setup Guide

How to configure and use the Finexus admin panel

Follow these steps to fully connect Finexus:

  1. Firebase Project
    Go to console.firebase.google.com → New Project → Finexus
  2. Enable Authentication
    Build → Authentication → Sign-in method → Email/Password → Enable
  3. Realtime Database Rules
    Go to Realtime Database → Rules and paste your secure rules.
  4. Google Drive Images
    Upload to Drive → Share → Anyone with link → Copy URL → Paste into image field
  5. Contact Form Messages
    Messages submitted on the website appear in the Messages section automatically.
Add New Property
